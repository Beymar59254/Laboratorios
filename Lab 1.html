Laboratorio #1

Introducción

El presente laboratorio tiene como propósito fundamental el desarrollo de aplicaciones web utilizando tecnologías modernas del ecosistema de Internet. En este primer laboratorio, se abordan tres ejercicios progresivos que permiten al estudiante comprender desde los conceptos básicos hasta la implementación de sistemas más complejos con validación cliente-servidor.

El laboratorio está estructurado para desarrollar una calculadora básica en tres versiones diferentes: la primera utilizando JavaScript puro (vanilla JS), la segunda empleando Bootstrap 5 y jQuery para un diseño responsivo y moderno, y la tercera implementa un sistema de login profesional.

Objetivos

  • Implementar una calculadora funcional utilizando JavaScript puro con validaciones completas.
  • Desarrollar interfaces responsivas utilizando Bootstrap 5 y jQuery.
  • Crear un sistema de autenticación seguro con PHP y Ajax.
  • Implementar validaciones tanto del lado del cliente como del servidor.
  • Utilizar mensajes de alerta elegantes con la librería SweetAlert2.
  • Aplicar buenas prácticas de programación y código limpio y comentado.

Desarrollo por Ejercicio

Ejercicio 1: Calculadora con JavaScript Puro

El primer ejercicio consiste en crear una calculadora funcional que realiza las operaciones básicas de suma, resta, multiplicación y división. La implementación utiliza HTML5 para la estructura y JavaScript vanilla para la lógica.

HTML5JavaScriptCSS3
Características:
  • Funciones JavaScript separadas para cada operación.
  • Validación de campos vacíos.
  • Validación de tipos de datos.
  • Validación para división entre cero.
  • Uso de parseFloat() para decimales.
  • Mensajes de error con alert().
Ejercicio 2: Calculadora con Bootstrap 5 y jQuery

El segundo ejercicio replica la funcionalidad del primero, pero con un enfoque moderno utilizando Bootstrap 5 para el diseño responsivo y jQuery para las validaciones.

Bootstrap 5jQuerySweetAlert2
Características:
  • Diseño moderno con tarjeta centrada.
  • Sistema de grid de Bootstrap.
  • Botones con colores diferenciados.
  • Animaciones suaves.
  • Validaciones con jQuery.
  • Mensajes con SweetAlert2.
  • Diseño responsivo.
Ejercicio 3: Sistema de Login Profesional

El tercer ejercicio implementa un sistema de autenticación completo con frontend y backend separado.

HTML5Bootstrap 5jQueryPHPAjax
Características:
  • Formulario con validación jQuery.
  • Comunicación mediante Ajax (POST).
  • Backend PHP con validación.
  • Protección contra inyección SQL básica.
  • Respuesta JSON.

Conclusiones

El laboratorio #1 de Programación en Internet ha permitido desarrollar competencias fundamentales en el desarrollo web moderno. A través de los tres ejercicios propuestos, se logró implementar desde una calculadora básica con JavaScript puro hasta un sistema de autenticación completo con PHP y Ajax. Los estudiantes pudieron comprender la diferencia entre el desarrollo del lado del cliente (frontend) y del servidor (backend), así como la importancia de las validaciones en ambos niveles.
