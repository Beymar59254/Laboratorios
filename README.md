# Laboratorio #1 - Programación en Internet

## Descripción

Este es el primer laboratorio de la materia de Programación en Internet. Consiste en 3 ejercicios donde se desarrollan diferentes habilidades de programación web.

## Ejercicios

### Ejercicio 1: Calculadora con JavaScript Puro
- Calculadora básica que hace suma, resta, multiplicación y división
- Usa HTML5 y JavaScript vanilla
- Validaciones para campos vacíos y división por cero
- Usa parseFloat() para números decimales

### Ejercicio 2: Calculadora con Bootstrap y jQuery
- Igual que el ejercicio 1 pero con diseño moderno
- Usa Bootstrap 5 para que se vea bien en celular
- Usa jQuery para las validaciones
- Mensajes nice con SweetAlert2

### Ejercicio 3: Sistema de Login
- Formulario de login con usuario y contraseña
- Usa Ajax para enviar datos sin recargar la página
- PHP en el servidor para validar
- Tiene protección básica contra inyección

**Credenciales:** admin / 123456

## Tecnologías

- HTML5
- CSS3
- JavaScript
- Bootstrap 5
- jQuery
- PHP
- SweetAlert2

## Cómo usarlo

### Para los ejercicios 1 y 2:
Solo abre el archivo index.html en cualquier navegador.

### Para el ejercicio 3:
Necesitas un servidor con PHP (como XAMPP). Pon la carpeta en htdocs y abre en el navegador.
