Laboratorio #1Programación en Internet

Introducción

El presente laboratorio tiene como propósito fundamental el desarrollo de aplicaciones web utilizando tecnologías modernas del ecosistema de Internet. En este primer laboratorio, se abordan tres ejercicios progresivos que permiten al estudiante comprender desde los conceptos básicos hasta la implementación de sistemas más complejos con validación cliente-servidor.

El laboratorio está estructurado para desarrollar una calculadora básica en tres versiones diferentes: la primera utilizando JavaScript puro (vanilla JS), la segunda empleando Bootstrap 5 y jQuery para un diseño responsivo y moderno, y la tercera implementa un sistema de login profesional con PHP y Ajax. Este enfoque permite al estudiante comparar las diferentes tecnologías y comprender sus ventajas y desventajas.

Todas las implementaciones siguen las mejores prácticas de desarrollo web, incluyen validaciones robustas tanto en el cliente como en el servidor, y presentan un diseño profesional adecuado para aplicaciones reales.

Objetivos

  • Implementar una calculadora funcional utilizando JavaScript puro con validaciones completas.
  • Desarrollar interfaces responsivas utilizando Bootstrap 5 y jQuery.
  • Crear un sistema de autenticación seguro con PHP y Ajax.
  • Implementar validaciones tanto del lado del cliente como del servidor.
  • Utilizar mensajes de alerta elegantes con la librería SweetAlert2.
  • Aplicar buenas prácticas de programación y código limpio y comentado.
  • Comprender el flujo de datos entre cliente y servidor mediante Ajax.

Desarrollo por Ejercicio

Ejercicio 1: Calculadora con JavaScript Puro

El primer ejercicio consiste en crear una calculadora funcional que realiza las operaciones básicas de suma, resta, multiplicación y división. La implementación utiliza HTML5 para la estructura y JavaScript vanilla para la lógica.

HTML5JavaScriptCSS3
Características implementadas:
  • Funciones JavaScript separadas para cada operación aritmética.
  • Validación de campos vacíos antes de realizar cálculos.
  • Validación de tipos de datos (verificación de números).
  • Validación especial para división entre cero.
  • Uso de parseFloat() para manejar números decimales correctamente.
  • Mensajes de error mediante alert().
  • Diseño básico pero ordenado y centrado en pantalla.
  • Código completamente comentado para fines educativos.

Ejercicio 2: Calculadora con Bootstrap 5 y jQuery

El segundo ejercicio replica la funcionalidad del primero, pero con un enfoque moderno utilizando Bootstrap 5 para el diseño responsivo y jQuery para la manipulación del DOM y las validaciones.

Bootstrap 5jQuerySweetAlert2
Características implementadas:
  • Diseño moderno con tarjeta centrada y bordes redondeados.
  • Sistema de grid de Bootstrap para botones organizados.
  • Botones con colores diferenciados para cada operación.
  • Animaciones suaves en botones y resultado.
  • Validaciones utilizando jQuery.
  • Mensajes de error y éxito elegantes con SweetAlert2.
  • Diseño completamente responsivo (adaptable a móviles).
  • Feedback visual en tiempo real.

Ejercicio 3: Sistema de Login Profesional

El tercer ejercicio implementa un sistema de autenticación completo con frontend y backend separado. Utiliza PHP para el procesamiento del lado del servidor y Ajax para la comunicación sin recargar la página.

HTML5Bootstrap 5jQueryPHPAjaxSweetAlert2
Características implementadas:
  • Frontend: Formulario con validación jQuery antes del envío.
  • Comunicación: Envío de datos mediante Ajax (POST).
  • Backend PHP: Recepción y validación de datos.
  • Seguridad: Validación de entradas, sanitización contra inyección SQL básica.
  • Respuesta: JSON con formato estructurado.
  • Credenciales: Simulación con usuario: admin, contraseña: 123456.
  • Estructura MVC: Código separado en index.html, script.js, styles.css, login.php.
  • Logging: Registro de intentos de acceso.

Conclusiones

El laboratorio #1 de Programación en Internet ha permitido desarrollar competencias fundamentales en el desarrollo web moderno. A través de los tres ejercicios propuestos, se logró implementar desde una calculadora básica con JavaScript puro hasta un sistema de autenticación completo con PHP y Ajax.

Los estudiantes pudieron comprender la diferencia entre el desarrollo del lado del cliente (frontend) y del servidor (backend), así como la importancia de las validaciones en ambos niveles para garantizar la seguridad y funcionalidad de las aplicaciones web. El uso de librerías como Bootstrap, jQuery y SweetAlert2 demuestra cómo estas herramientas facilitan el desarrollo rápido y profesional de interfaces web.
